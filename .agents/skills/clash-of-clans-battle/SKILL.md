# 部落冲突自动战斗脚本

## 概述

此技能用于自动化《部落冲突》(Clash of Clans) 的战斗流程，包括进入进攻界面、搜索对手、确认战斗、部署兵力等操作。

---

## 核心流程

### 阶段一：进入进攻界面

```
1. [截屏] 查看当前屏幕
2. [识别] 查找带有"进攻!"文字的地图图标（通常在屏幕左下区域）
   - 图标特征：地图样式，带有"进攻!"文字
3. [点击] 点击进攻图标
4. [等待] 1-2秒等待界面切换
5. [截屏] 确认已进入进攻选择界面
```

**进攻图标位置参考**：
- 通常位于屏幕左下角区域
- 归一化坐标范围：x [50, 200], y [850, 950]
- 图标特征：地图样式图标，带有"进攻!"或"Attack!"文字

### 阶段二：搜索对手

```
1. [截屏] 查看当前界面
2. [识别] 查找"搜索对手"按钮
   - 按钮特征：带有"搜索对手"文字
3. [点击] 点击搜索对手按钮
```

**搜索对手按钮位置参考**：
- 通常位于屏幕左侧

### 阶段三：确认进攻

```
1. [截屏] 查看兵种配置界面
2. [识别] 查找"进攻!"按钮（带交叉双剑图标）
   - 按钮特征：带有"进攻!"文字，有两把交叉的剑
3. [点击] 点击进攻按钮
4. [等待] 2-3秒等待进入战斗
5. [截屏] 确认是否进入战斗界面
```

**进攻按钮位置参考**：
- 通常位于屏幕右下角
- 图标特征：带有"进攻!"文字，有两把交叉的剑

### 阶段四：战斗界面确认

```
1. [检测] 检查是否有倒计时界面出现
2. [检测] 检查是否有"结束战斗"按钮
3. [确认] 如果同时存在倒计时和结束战斗按钮，则确认已进入战斗界面
```

**战斗界面特征**：
- 存在倒计时显示
- 存在"结束战斗"按钮
- 屏幕底部显示兵力图标栏

### 阶段五：兵力部署

```
1. [截屏] 查看战斗界面
2. [识别] 查找兵力选择图标（带有"x数字"的图标）
   - 位置：屏幕底部
   - 排除"x0"的图标（表示没有该兵力）
   - 优先选择数量较多的兵力
3. [点击] 选择兵力图标
4. [等待] 0.5秒确认选中
5. [截屏] 确认兵力选中状态
6. [检测] 检查兵力是否被选中（白框高亮效果）
   - 选中特征：兵种图标周围有白框，呈现突出/高亮效果
   - 未选中：无白框，图标平淡
7. [确认] 只有检测到白框选中效果后，才能进行部署
8. [识别] 识别村庄范围内左侧或右侧的小树位置
   - 小树特征：村庄内的树木障碍物（不是包围村庄的树林）
   - 位置：村庄范围内，通常在建筑区域的左侧或右侧
   - 注意：不是地图最外围的树林
9. [部署] 在小树旁边的绿色草地区域长按投放兵力
   - 投放区：小树旁边的纯绿色草地（无红色阴影）
   - 投放方式：长按3秒投放兵力
   - 投放点：小树两侧或附近的绿色草地区域
   - 多方向：可在左侧和右侧小树分别投放，形成多线进攻
10. [重复] 选择其他兵力并继续部署
```

**兵力图标识别**：
- 位置：屏幕底部
- 特征：带有"x数字"表示数量，如"x20"、"x5"
- **排除条件**：显示"x0"的图标表示无可用兵力

**兵力选中状态识别**：
- **选中特征**：兵种图标周围有**白框**包裹，呈现突出/高亮效果
- **未选中特征**：图标无白框，呈现平淡状态
- **重要**：只有确认白框选中效果后，才能进行兵力部署

**投放区域识别**：
- **小树特征**：村庄内的树木障碍物，位于建筑区域的左侧或右侧（注意：不是地图外围的树林）
- **有效投放区**：小树旁边的纯绿色草地（没有红色阴影的区域）
- **禁止投放区**：淡红色阴影区域及其深红色边框内部
- **判断方法**：
  1. 扫描村庄范围内（非地图边缘）的左侧和右侧区域
  2. 识别村庄内小树的绿色树冠和树干特征
  3. 选择小树旁边或两侧的绿色草地作为投放点

**部署操作要点**：
1. 先点击兵力图标
2. 截屏确认白框选中效果
3. 识别村庄范围内左侧或右侧的小树位置（不是外围树林）
4. 在小树旁边的绿色草地区域**长按3秒**投放兵力
5. 可选择左侧和右侧小树分别投放，形成多线进攻
6. 每次投放后观察是否成功

**投放位置参考**：
- 投放点应在小树旁边的纯绿色草地上
- 小树是指村庄内的树木障碍物，不是地图外围的树林
- 投放方式：长按3秒投放兵力
- 避免点击在淡红色阴影区域内或深红色边框线上
- 可在小树两侧或附近区域投放，形成多点进攻

## 时间控制规范

### 界面切换等待

| 操作 | 等待时间 | 说明 |
|------|----------|------|
| 点击进攻图标后 | 1-2秒 | 等待进攻界面加载 |
| 点击搜索对手后 | 3-5秒 | 等待匹配对手 |
| 点击进攻按钮后 | 2-3秒 | 等待进入战斗界面 |
| 选择兵力后 | 0.5秒 | 确认选中状态 |
| 部署兵力时 | 3秒 | 长按投放兵力 |
| 部署兵力后 | 1-2秒 | 等待兵力下落 |


## 异常处理机制

### 1. 操作失败重试

```
操作失败时的重试流程：
1. 记录失败原因
2. 等待 1 秒
3. 重新截屏确认当前状态
4. 尝试替代操作方案
5. 最多重试 3 次
6. 3 次失败后跳过当前步骤或终止任务
```

### 2. 超时检测

```
每个阶段的超时设置：
- 进攻图标识别：10秒超时
- 搜索对手匹配：30秒超时
- 战斗确认：15秒超时
- 兵力部署：60秒超时

超时处理：
1. 记录超时日志
2. 尝试返回上一界面
3. 重新开始当前阶段
```

### 3. 执行日志记录

每次关键操作都应记录：
```
[时间戳] 操作: 点击进攻图标
[时间戳] 位置: [120, 900]
[时间戳] 结果: 成功/失败
[时间戳] 等待: 1500ms
[时间戳] 状态: 已进入进攻界面
```

## 注意事项

1. **区分两个进攻按钮**：
   - 主界面的进攻图标：地图样式，在左下角
   - 战斗确认的进攻按钮：带交叉双剑，在右下角
2. **战斗界面确认**：必须同时检测到倒计时和结束战斗按钮
3. **兵力选中确认**：必须检测到白框选中效果才能部署兵力
4. **投放区域确认**：
   - 小树是指村庄内的树木障碍物，不是地图外围的树林
   - 在小树旁边的纯绿色草地投放兵力
   - 投放方式：长按3秒投放
   - 点击位置：小树两侧或附近的绿色草地
   - 禁止点击淡红色阴影区域内或深红色边框线上
5. **兵力检查**：只选择"x数字"大于0的兵力图标
6. **多线部署**：在左侧和右侧小树分别投放兵力，形成多线进攻
7. **网络延迟**：搜索对手时可能需要更长的等待时间

## 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| 找不到进攻图标 | 滑动屏幕或检查是否在主界面 |
| 搜索对手超时 | 检查网络连接，重新尝试 |
| 未检测到战斗界面 | 确认是否成功点击进攻按钮 |
| 兵力图标显示x0 | 切换到其他可用兵力 |
| 兵力未选中（无白框） | 重新点击兵力图标，等待后截屏确认 |
| 找不到小树 | 检查村庄范围内（非外围树林），或选择其他投放参照物 |
| 投放位置无效 | 确保长按小树旁边的纯绿色草地3秒，不要在淡红色区域内或边框线上 |
| 点击后无响应 | 增加等待时间，重新截屏确认 |
| 无法进入战斗 | 检查是否有点击正确的进攻按钮（带交叉双剑） |
| 投放兵力失败 | 确认白框选中效果，确认长按3秒在小树旁边的纯绿色草地 |
