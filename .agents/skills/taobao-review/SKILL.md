# 淘宝商品评论操作

## 依赖技能

本技能依赖 **text-input** 技能来辅助完成文字输入操作：
- 检查输入法状态（ADB Keyboard {ON}）
- 清空输入框原有内容
- 输入评论文字

**重要**：在执行评论任务前，必须先加载 text-input 技能，然后再进行评论操作。

## 核心原则

1. **评论内容必须根据商品类型编写真实、有意义的评价**，禁止乱写无关内容。每条评论需20字以上。
2. **不能上传图片**，只进行文字评论，不添加任何图片或视频。
3. **每次点击操作后必须截图确认界面状态**，确保操作成功后再继续下一步。

## 评论内容规范

### 根据商品类型编写评论

评论内容应与商品类型相关，包含以下要素：
- 商品质量/做工
- 使用体验
- 物流/包装
- 是否推荐

### 评论内容要求

```
字数要求：20字以上（约15-30个字为宜）
内容要求：与商品类型相关，真实可信
禁止行为：复制粘贴相同内容、乱写无关文字
```

## 操作流程

### 整体流程

```
1. [截屏] 查看待评论列表
2. [循环] 遍历列表中的每个商品：
   ├─ 点击商品进入评论页面
   ├─ 选择5星好评
   ├─ 编写评论内容（根据商品类型）
   ├─ 提交评论
   └─ 返回列表，继续下一个
3. [完成] 确认所有商品已评论
```

### 单个商品评论流程

```
1. [截屏] 查看当前商品信息，识别商品类型
2. [定位] 找到5星下方的大块空白区域
3. [点击] 点击该空白区域激活输入框
4. [截屏] 确认输入框已激活，检查界面状态
5. [检查] 输入法是否已激活（ADB Keyboard {ON}）
   ├─ 未激活 → 再次点击输入区域
   └─ 已激活 → 继续下一步
6. [输入] 根据商品类型输入评论内容（20字以上）
7. [选择] 确认5星好评已选中
8. [提交] 点击提交按钮
9. [截屏] 确认提交结果，查看是否成功
10. [确认] 等待评论成功提示或返回列表
```

## 详细步骤说明

### 步骤一：识别商品类型

通过截屏分析商品图片、名称、描述等信息，判断商品属于哪个类别：
- 观察商品主图
- 阅读商品名称
- 查看商品分类标签

### 步骤二：选择星级

淘宝评论默认为5星好评：
1. 找到星级选择区域
2. 确认已选择5颗星（通常星星会高亮显示）
3. 如未选中，点击第5颗星

### 步骤三：定位评论输入框

评论输入框的位置特征：
- 位于5星评价区域的**正下方**
- **中间有一大块空白区域**，用于输入评论内容
- 点击该空白区域即可激活输入框
- 激活后会弹出键盘，显示输入光标

### 步骤四：编写评论

按照文本输入 skill 的规范：
1. 检查输入法状态
2. 清空原有内容（如有）
3. 输入根据商品类型编写的评论

### 步骤四：提交评论

1. 点击"发表评论"或"提交"按钮
2. 等待成功提示
3. 自动返回待评论列表

## 常见错误

| 错误行为 | 正确做法 |
|---------|---------|
| 所有商品使用相同评论 | 根据商品类型编写不同评论 |
| 评论内容与商品无关 | 评论应涉及商品质量、体验等 |
| 评论字数不足20字 | 确保评论内容充实，20字以上 |
| 未确认星级就提交 | 先确认5星已选中再提交 |
| 评论内容乱写 | 根据商品类型编写真实评价 |
| 忽略输入法状态 | 先检查输入法是否激活 |
| 找不到评论输入框 | 找5星正下方的大块空白区域 |

## 注意事项

1. **评论真实性**：评论内容应真实可信，不要使用过于夸张的语言
2. **商品识别**：仔细识别商品类型，避免评论内容与商品不符
3. **网络延迟**：提交评论后等待成功提示，避免重复提交
4. **列表遍历**：注意列表可能有滚动，需要滑动查看更多待评论商品
5. **截图确认**：每次点击操作后都要截图确认界面状态，验证操作是否成功
